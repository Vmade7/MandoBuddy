

# **🎨 Figma 设计 Prompt 文档 — MandoBuddy**

## **🧭 一、设计总原则（适用于 Web & iOS）**



```
整体风格：
- 现代简约 + 轻国风
- 主色调：柔和黄色 (#F6E8C3) + 暖橙 (#F9B24E)
- 辅色：米白 (#FFF9F0) / 中灰 (#707070)
- 字体：简洁易读（Web: Noto Sans / iOS: SF Pro）
- 设计节奏稳定、层次清晰、易用性优先
```

视觉系统：



- 模块卡片化布局
- 大圆角卡片 + 辅以轻中式纹样点缀
- 图标统一线性风格 + 简约文化图标元素
- 动效柔和渐变、按钮反馈清晰

## **📌 二、Web 端设计 Prompt**





------





### **1️⃣ 首页 Home（Web）**



```
页面名：Web_Home
布局结构：
- 顶部导航：Logo | 学习页 | 文化彩蛋 | 个人中心
- Banner 区：今日学习进度 + 推荐任务
- 主内容 Grid：拼音练习、场景练习、发音评测、每日文化彩蛋
- 底部浮层：动态节日专题提示卡
```

设计要求：



- 横向 Grid 卡片整齐排列，留白充足
- 模块入口 ICON + 文案清晰
- 节日卡片应支持悬停动效
- 大屏优先展示进度概览





------





### **2️⃣ 学习模块 — 场景学习 Web**



```
页面名：Web_Scene_Learning
组件：
- 顶部 Banner：场景名称 + 主题色条
- 场景词汇滑动区：左右滑动
- 短句跟读区：卡片式文本 + 播放/录音按钮
- 对话练习卡片：对话气泡 + 角色头像
- 底部入口：AI 评测按钮 + 文化彩蛋链接
```

交互要求：



- 鼠标移入播放按钮高亮
- 词汇卡支持左右键盘切换
- 跟读区固定操作栏悬浮底部





------





### **3️⃣ 发音评测界面 Web**



```
页面名：Web_Pronunciation
布局：
- 标题区：目标句 & 拼音
- 录音交互区：波形动画 + 录音按钮
```

反馈交互：

## **⭐ 四、平台差异化设计注意事项**



| **项目** | **Web 端**     | **iOS 端**     |
| -------- | -------------- | -------------- |
| 导航     | 顶栏 + 超链接  | Tab 底栏       |
| 交互     | 鼠标 + 键盘    | 触控手势       |
| 屏幕密度 | 大屏优先       | 小屏优先       |
| 动效节奏 | 交互反馈更细腻 | 手势驱动更流畅 |
| 卡片表现 | 层级分明       | 触控易触达     |
