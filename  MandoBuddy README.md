# ** MandoBuddy README**





## **English Version**





### **Project Overview**





MandoBuddy is a lightweight web-based Chinese learning platform designed for beginner learners. It focuses on providing users with an engaging, interactive experience to improve their Chinese language skills, while also introducing them to Chinese culture through cultural “Easter eggs.” The platform features pronunciation exercises, scene-based dialogues, AI-powered pronunciation evaluation, and daily cultural content.





### **Core Features**





1. **Pronunciation Practice**:

   

   - Practice pinyin with a focus on initials, finals, and tones.
   - AI-powered real-time pronunciation feedback.

   

2. **Scene-Based Learning**:

   

   - Practical vocabulary and phrases in real-life scenarios like dining, travel, and shopping.
   - Interactive dialogues with real-life context.

   

3. **Daily Cultural “Easter Eggs”**:

   

   - Learn about Chinese festivals, customs, and traditions through short content pieces.
   - Cultural vocabulary learning and comparison with Western traditions.

   

4. **Progress Tracking**:

   

   - Visual progress bars to track daily and cumulative learning.
   - Unlock badges and achievements for continuous learning.

   







### **Technologies Used**





- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: FastAPI, Supabase (for user management and data storage)
- **AI Evaluation**: Web Audio API, MediaRecorder (for pronunciation feedback)
- **Deployment**: Vercel (Frontend), Docker (Backend)







### **Getting Started**





1. Clone the repository:

   git clone https://github.com/your-repo/MandoBuddy.git

2. Install dependencies:

   

   - Frontend: npm install
   - Backend: pip install -r requirements.txt

   

3. Run the development server:

   

   - Frontend: npm run dev
   - Backend: uvicorn app:app --reload

## **中文版**







### **项目概述**





MandoBuddy 是一款为中文初学者设计的轻量级 Web 中文学习平台。它专注于为用户提供互动、富有趣味的学习体验，同时通过文化“彩蛋”将中国传统文化融入其中。平台功能包括发音练习、情景对话、AI 发音评估和每日文化内容。





### **核心功能**





1. **拼音练习**：

   

   - 针对声母、韵母和声调进行拼音练习。
   - 提供 AI 实时发音反馈。

   

2. **场景学习**：

   

   - 学习餐饮、旅游、购物等日常生活场景中的词汇和短语。
   - 与真实场景对话互动。

   

3. **每日文化彩蛋**：

   

   - 通过简短内容了解中国节日、习俗和传统。
   - 学习相关文化词汇，并与西方传统进行对比。

   

4. **进度追踪**：

   

   - 可视化进度条，追踪每日和累计学习进度。
   - 解锁徽章和成就，激励持续学习。

   







### **技术栈**





- **前端**：React, Vite, Tailwind CSS
- **后端**：FastAPI, Supabase（用户管理和数据存储）
- **AI 评估**：Web Audio API, MediaRecorder（发音反馈）
- **部署**：Vercel（前端），Docker（后端）







### **开始使用**





1. 克隆项目库：

   git clone https://github.com/vmade7/MandoBuddy.git

2. 安装依赖：

   

   - 前端：npm install
   - 后端：pip install -r requirements.txt

   

3. 运行开发服务器：

   

   - 前端：npm run dev
   - 后端：uvicorn app:app --reload
