## 六、用户与场景分析

### （一）用户画像（Personas）

**🎓 类型一：零基础成人学习者**
- 典型代表：Emma，28岁，美国白领
- 核心特征：工作繁忙、利用碎片时间学习、对中文好奇但存在畏难情绪
- 学习目标：3个月内掌握基础问候、点餐、问路等生存中文
- 痛点诉求：害怕语法复杂、担心发音不准无人纠正、难以坚持长期学习

**👦 类型二：海外青少年/华裔二代**
- 典型代表：Lucas，14岁，华裔二代
- 核心特征：喜欢游戏化内容、短视频形式、对中国文化有天然兴趣
- 学习目标：能与家人简单中文交流、看懂中文短视频字幕
- 痛点诉求：拒绝枯燥背单词、需要趣味互动、渴望即时正向反馈

**✈️ 类型三：短期来华商务/旅居人士**
- 典型代表：Marco，35岁，意大利商务人士
- 核心特征：2-3个月后出差中国、时间紧迫、学习目标明确
- 学习目标：掌握机场、酒店、餐厅、打车等场景高频短句
- 痛点诉求：需要即学即用、发音准确、支持离线复习

**🌏 类型四：对华文化爱好者**
- 典型代表：Sofia，22岁，西班牙大学生
- 核心特征：喜欢中国影视/美食/传统文化、自学能力强
- 学习目标：看懂简单中文内容、旅行时能进行基础沟通
- 痛点诉求：希望学习过程融入文化故事、内容不枯燥、有成就感激励

### （二）典型使用场景（User Scenarios）

**场景1：碎片化晨间学习（零基础成人）**
- 时间：工作日早上7:30-8:00
- 场景：通勤地铁上，手机浏览器打开 MandoBuddy
- 用户行为：
  1. 系统自动续学昨日「拼音声调」课程
  2. 跟读3个新韵母，AI实时评分85分，重读后提升至92分
  3. 完成5道趣味选择题巩固，解锁「问候语」新模块
  4. 收藏"你好吗？"短句，加入今日复习清单
- 核心价值：30分钟完成「学-练-测-存」闭环，无压力入门

**场景2：亲子共学互动（华裔青少年）**
- 时间：周末下午，家庭平板设备
- 场景：孩子学习，家长旁协助
- 用户行为：
  1. 进入「春节文化」主题模块，看动画学"新年快乐"
  2. 孩子跟读发音，AI给出"👍 声调很准！"鼓励反馈
  3. 完成「贴春联」互动小游戏，拖拽汉字配对拼音
  4. 生成学习成就卡片，一键分享给家庭群
- 核心价值：文化+语言融合，游戏化降低学习抵触，增强家庭互动

**场景3：出差前紧急突击（商务人士）**
- 时间：出差前1周，每晚20分钟
- 场景：酒店房间，笔记本电脑+耳机
- 用户行为：
  1. 选择「商务生存包」快捷学习路径
  2. 聚焦学习：打车/点餐/酒店入住/数字价格 4大场景
  3. 开启「发音强化模式」，重点纠正常见声调错误
  4. 导出「离线复习卡」，保存至手机随时查看
- 核心价值：精准打击高频场景，短期高效建立沟通信心

**场景4：文化探索式学习（爱好者）**
- 时间：晚间休闲时段，平板+耳机
- 场景：放松状态下兴趣驱动学习
- 用户行为：
  1. 浏览「文化彩蛋」栏目，点击「中秋节」主题
  2. 看30秒动画了解月饼文化，同步学习"月饼""团圆"等词汇
  3. 尝试用刚学的短句"我想吃月饼"跟读，获得发音反馈
  4. 收藏文化知识点，生成「我的中文文化手账」
- 核心价值：以文化为钩子，自然带动语言学习，提升持续动力

### （三）用户旅程地图（简化版）

**阶段1：认知阶段**
- 用户搜索"learn Chinese online"等关键词
- 看到 MandoBuddy 轻量网页广告或社交媒体推荐

**阶段2：体验阶段**
- 无需注册，游客模式体验1节拼音课
- 界面英文引导，30秒内快速上手

**阶段3：学习阶段**
- 邮箱快捷注册，同步学习进度至云端
- 按推荐路径学习「生存中文」模块
- AI发音反馈+趣味练习即时巩固

**阶段4：巩固阶段**
- 系统智能推送错题复习+文化彩蛋
- 学习成就徽章激励持续学习

**阶段5：分享阶段**
- 生成个性化学习报告/文化手账
- 一键分享社交媒体，邀请好友加入

## 七、信息架构（Information Architecture）

### （一）整体站点结构

**🏠 首页 Home**
- 核心价值主张：3秒理解产品定位（轻量/有趣/能开口）
- 快速开始入口：游客体验课 / 立即注册
- 学习路径推荐：零基础 / 商务 / 文化兴趣 三类入口
- 用户见证 + 文化彩蛋预览

**📚 学习中心 Learn**
- 学习路径选择（按目标分流）
  - 零基础入门：拼音→汉字→短句 阶梯式学习
  - 生存中文：问候/饮食/交通/购物 四大场景
  - 商务速成：职场/数字/预约/礼仪 实用内容
  - 文化兴趣：节日/美食/影视/习俗 主题学习

- 课程模块（阶梯式展开）
  - Level 1 拼音基础
    - 声母学习：图文讲解+标准音频+跟读练习
    - 韵母学习：口型演示+对比辨析+即时反馈
    - 声调训练：四声+轻声 动态演示+跟读评测
    - 拼音综合：拼读练习+趣味闯关
  
  - Level 2 基础词汇
    - 场景分类：问候/数字/颜色/食物等主题
    - 词汇卡片：图片+发音+英文释义+例句+收藏
    - 词汇游戏：听音选词/图片配对/拼字挑战
  
  - Level 3 实用短句
    - 场景对话：音频+中英字幕+角色扮演+逐句跟读
    - 句型拆解：主谓宾可视化+语法点轻量提示
    - 情景模拟：选择回应+发音练习+即时反馈

- 发音训练（独立强化模块）
  - AI跟读评测：单词/短句/自由朗读 多模式支持
  - 常见错误库：中外发音对比+针对性纠正建议
  - 进步轨迹：发音评分趋势+薄弱点智能提醒

- 趣味练习
  - 随堂小测：学完即练，题型简单易懂
  - 单元挑战：闯关式测试，解锁成就徽章
  - 错题本：自动收录+智能复习提醒+一键重练
  - 成就系统：徽章收集+进度奖励+分享激励

**🌏 文化角落 Culture**
- 文化彩蛋：轻量科普（节日/礼仪/冷知识），中英对照
- 文化画廊：高清图片+短文案+相关词汇联动
- 微视频：30-60秒文化场景短片，配字幕+词汇标注
- 文化小测：学完文化点即时巩固，趣味问答形式

**👤 个人中心 Profile**
- 学习数据看板：进度条+学习时长+掌握度+发音评分趋势
- 我的收藏：词汇/短句/文化内容 分类管理+一键复习
- 错题本：按模块/时间/错误类型 筛选+导出复习卡
- 设置中心：语言切换/通知偏好/数据导出/账号管理
- 成就墙：徽章展示+学习里程碑+分享卡片生成

**ℹ️ 帮助与支持 Help**
- 常见问题：注册/发音/进度/支付 等高频问题解答
- 联系我们：反馈入口+客服邮箱+社区链接
- 语言切换：英文/西语/法语/日语 等多语言支持

**🔐 管理后台（独立域名）**
- 内容管理 / 用户管理 / 数据统计 / 系统配置

```

  ↓

  ↓
进入课程模块（列表页展示章节+进度）

  ↓

  ↓
```

### （三）关键交互原则

- **3次点击原则**：任何核心学习内容，从首页进入不超过3次点击
- **双语无缝切换**：界面操作指令始终英文（或多语言），学习内容中英对照展示
- **进度可视化**：每页顶部显示当前模块进度条+本节完成状态，降低认知负担
- **即时反馈**：跟读/答题后0.5秒内给出结果，减少等待焦虑，增强学习动力
- **断点续学**：关闭页面后重新进入，自动定位上次学习位置，支持多端同步
- **轻量加载**：首屏核心内容优先加载，音频/图片懒加载，保障海外访问速度

## 八、功能清单（Feature List）

### （一）前台用户端功能

**🔹 用户账户体系**
- 邮箱快捷注册/登录（P0）：支持密码登录+验证码登录，流程简化至3步内
- 游客模式（P0）：无需注册即可体验1-2节基础课程，学习数据本地暂存，注册后自动同步
- 第三方登录（P1，二期）：支持Google/Facebook/Apple ID 快速授权登录
- 多语言界面切换（P1）：首期支持英文界面，二期扩展西语/法语/日语/韩语
- 学习数据看板（P0）：可视化展示学习进度、累计时长、知识点掌握度、发音评分趋势
- 收藏/错题管理（P0）：支持按模块/时间/类型筛选，一键复习，支持导出离线复习卡
- 成就系统（P1）：徽章解锁动画、进度奖励提示、生成分享卡片支持社交媒体传播

**🔹 核心学习功能**
- 拼音学习模块（P0）
  - 声母/韵母/声调图文讲解：配动态口型图、标准音频、英文发音提示
  - 跟读+AI实时评测：WebRTC录音+云端ASR识别+评分+纠音建议（如"第三声下沉不够"）
  - 拼音对比练习：易混音（如j/q/x）专项训练，配波形图对比

- 词汇学习模块（P0）
  - 场景化词汇卡片：图片+发音+英文释义+例句+收藏按钮，支持点击发音/慢速播放
  - 词汇联想记忆：同主题词汇关联展示（如"餐厅"场景关联"菜单/筷子/买单"）
  - 词汇小游戏（P1）：听音选词/图片配对/拼字挑战等轻量互动，即时反馈+积分奖励

- 短句学习模块（P0）
  - 场景对话展示：音频+中英字幕+角色切换+逐句跟读，支持单句循环/变速播放
  - 句型轻量拆解：主谓宾可视化标注，语法点用英文简短提示（避免术语堆砌）
  - 情景模拟练习：选择回应+发音练习+即时反馈，模拟真实对话场景

- 发音训练模块（P0）
  - AI跟读评测：支持单词/短句/自由朗读，评分维度包含准确度/流利度/声调
  - 常见发音错误库：针对英语/西语/法语母语者常见错误，提供对比音频+纠正技巧
  - 发音进步轨迹：生成周/月发音评分报告，标注提升点+待强化项

**🔹 练习巩固功能**
- 随堂小测（P0）：学完即练，题型包含选择题/填空题/跟读题，即时显示结果+解析
- 单元挑战（P1）：闯关式测试，通关解锁新模块+成就徽章，增强游戏化体验
- 错题本（P0）：自动收录答错题目，支持按错误类型/时间筛选，一键重练+智能复习提醒
- 复习计划（P1）：基于艾宾浩斯曲线，系统自动推送复习内容，支持手动调整

**🔹 文化拓展功能**
- 文化彩蛋（P0）：轻量科普卡片（节日/礼仪/冷知识），中英对照+相关词汇联动
- 文化画廊（P1）：高清图片+短文案+音频解说，点击词汇可跳转学习
- 微视频（P1）：30-60秒文化场景短片，配字幕+词汇标注+跟读入口
- 文化小测（P1）：趣味问答形式巩固文化知识点，答对解锁彩蛋内容

**🔹 体验优化功能**
- 离线复习卡（P1）：支持将收藏内容/错题导出为PDF/图片，方便无网环境复习
- 学习提醒（P1）：可设置每日学习提醒（邮件/浏览器通知），支持自定义时间
- 分享激励（P1）：生成学习报告/成就卡片，一键分享社交媒体，邀请好友得奖励
- 多端同步（P0）：学习进度/收藏/错题 云端同步，支持电脑/平板/手机无缝切换

### （二）后台管理端功能

**🔹 内容管理**
- 课程资源管理（P0）：支持新增/编辑/删除 拼音/词汇/短句/文化内容，富文本编辑器+多媒体上传
- 内容版本控制（P1）：支持课程内容历史版本回溯，便于迭代优化
- 多语言内容配置（P1）：同一学习内容配置多语言界面文案，支持动态切换

**🔹 用户管理**
- 用户列表查看（P0）：支持按注册时间/学习进度/活跃状态 筛选用户
- 用户数据详情（P0）：查看单个用户学习轨迹、发音评分、错题分布等
- 账号操作（P0）：支持禁用/启用账号、重置密码、数据导出（合规前提）

**🔹 数据统计**
- 学习行为分析（P0）：日活/留存/完课率/平均学习时长 等核心指标看板
- 内容效果分析（P1）：各课程模块的完成率、错题率、用户评分，辅助内容优化
- 发音数据洞察（P1）：聚合分析用户发音薄弱点，指导AI模型迭代+教学内容调整

**🔹 系统配置**
- 权限管理（P0）：角色分级（超级管理员/内容编辑/客服），操作权限精细化控制
- 审核配置（P0）：用户生成内容（如评论）的敏感词过滤+人工审核流程配置
- 系统监控（P1）：服务器状态、接口响应时间、错误日志 实时监控+告警

---

### （三）通用能力/技术支撑

**🔹 AI能力集成**
- 语音识别（ASR）（P0）：支持多口音中文识别，针对非母语者优化，响应时间<1秒
- 发音评测引擎（P0）：多维度评分（准确度/流利度/声调），提供可理解的纠音建议
- 智能推荐（P1）：基于用户学习行为，动态调整学习路径+复习内容

**🔹 性能与兼容性**
- 全球加速（P0）：CDN部署+海外节点优化，保障欧美/东南亚等地区访问速度
- 响应式布局（P0）：一套代码适配电脑/平板/手机，核心操作区域针对触屏优化
- 浏览器兼容（P0）：支持Chrome/Firefox/Safari/Edge 主流浏览器最新版本

**🔹 安全与合规**
- 数据加密（P0）：用户账号、学习数据 传输+存储全程加密，符合GDPR等海外合规要求
- 内容审核（P0）：学习资源+用户内容 双重审核机制，确保无违规信息
- 隐私保护（P0）：明确隐私政策，支持用户数据导出/删除请求，最小化数据采集

**🔹 可扩展性设计**
- 模块化架构（P1）：学习模块/练习题型/文化内容 支持插件式扩展，降低迭代成本
- 多语言框架（P1）：界面文案、学习内容 分离配置，便于快速拓展新语言市场
- API开放预留（P2）：核心能力（如发音评测）预留API接口，支持未来对接第三方平台

---

> 💡 优先级说明：P0=首期必做，保障核心体验；P1=二期优化，提升竞争力；P2=长期规划，探索创新

以上为 MandoBuddy 完整需求分析，聚焦「轻量化、趣味化、低门槛」核心定位，精准匹配海外零基础中文学习者的真实需求，为产品设计与开发提供清晰指引。
